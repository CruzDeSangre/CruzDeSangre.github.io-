⌘

ChatGPT

  • Web search thinks significantly longer (can easily take 7-10 min vs Gemini's 1-3 min)
  • Easy switching between different personalities
  • Can connect to a limited selection of websites, as well as to a web version of Photoshop, though tool calls can be inconsistent
  • Offers integration with Github, through the Codex site, as well as a Visual Studio Code extension
✦

Gemini

  • Deep research produces longer, more detailed investigations
  • Nanobanana Pro offers faster image generation than GPT Image 1.5. Its images also appear more realistic (subjectively). However, it must be taken into consideration that Google tends to throttle the daily limit of image generation requests when a new model or AI tool is released
  • Native Android integration (iOS confirmed to come soon), though it can be over-eager with tool calls
  • Offers uses of Claude Opus 4.5 and Claude Sonnet 4.5 through the Antigravity app
